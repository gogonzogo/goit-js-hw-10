Overview: This is a simple country lookup app that uses the REST Countries API to display information about a country.
It also uses the Mapbox API to display a map of the country.

Step 1: Enter a country name in the search box.
NOTE: As a list of countries populates you can, finish typing or click on the country from the list to display the country's information and find it on the map.

Step 2: Click on the plane icon to "fly" to the country for a closer look.

NOTE: The map also supports click to drag and scroll to zoom.

Happy exploring!
